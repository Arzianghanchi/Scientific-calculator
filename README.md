# Scientific-calculator
Scientifi Calculator
   

    This is a simple scientific calculator program written in Python.

    ## Features

    - Addition
    - Subtraction
    - Multiplication
    - Division
    - Square Root
    - Power

    ## Usage

    1. Run the `main.py` file.
    2. Choose an operation from the menu.
    3. Enter the required numbers.
    4. View the result.

    ## Requirements

    - Python 3.x
